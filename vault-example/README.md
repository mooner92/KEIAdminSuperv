# vault-example — 합성 예시 볼트 (실데이터 없음)

> ⚠️ 이 폴더는 **포트폴리오용 합성 예시**입니다. **실제 KEI 규정은 단 한 줄도 들어있지 않습니다.**
> 실제 운영 볼트(`KEI-행정가이드/`, 변환된 규정 원문)는 내부 전용이라 공개 레포에서 제외되며, Syncthing으로만 동기화됩니다. (루트 [SECURITY.md](../SECURITY.md) 참조)

이 시스템(KEI 행정 가이드/비서)이 **어떤 구조의 마크다운 볼트**를 먹는지 "구조만" 보여주기 위한 예시입니다. 모든 규정명·조문·금액·기관명은 가상입니다.

## 구조
| 폴더 | 역할 | 예시 |
|---|---|---|
| `20_규정원문/` | 규정 원문층(진실원천) | `9900_샘플규정.md` (9900 = 실제 번호 아님) |
| `10_업무가이드/` | 가치층(쉬운 설명) | 위키링크로 규정 인용 |
| `30_용어집/` | 개념 1개 = 노트 1개 | |
| `90_관리/_templates/` | 프론트매터 템플릿 3종 | |

실제 구조·규약(프론트매터 스키마, 제N조 청킹, 분류 1000~7999)은 [docs/03-content-model.md](../docs/03-content-model.md) 참조.
